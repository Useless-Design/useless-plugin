 按钮用来触发一些操作
## 演示

```demo
basic.vue
```

## API

### Button Props

| Name | Type | Default | Description | Vision |
| --- | --- | --- | --- | --- |
| attr-type | `'button' \| 'submit' \| 'reset'` | `'button'` | 按钮的 DOM 的 `type` 属性 |  |

### ButtonGroup Props

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| vertical | `boolean` | `false` | 组内按钮的排列方式 |

### Button Slots

| Name    | Default | Description                              |
| ------- | ------- | ---------------------------------------- |
| default | `()`    | 按钮的内容 |
